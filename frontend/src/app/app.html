The interface is intentionally reviewer-oriented: it shows the backend lifecycle instead of hiding the work behind a polished contact form.

  1. Validate Pydantic schemas reject unknown fields and invalid contact data.
  2. Rate limit Combined IP and email windows block spam before AI or email work.
  3. Analyze OpenAI-compatible classification enriches the owner notification.
  4. Persist Submissions, metrics, rate-limit state and outbox entries live in JSON storage.
  5. Deliver SMTP sends both emails; local outbox queues both when SMTP fails.
